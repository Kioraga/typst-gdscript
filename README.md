# codegds

GDscript syntax highlighting for [Typst](https://typst.app/).

![](example.png)

## Installation

````typst
#import "@preview/codeforth:0.1.0": forth-syntax
#set raw(syntaxes: forth-syntax)

```forth
: fun 1 2 + ;
```
````
